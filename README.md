# 🖥️ Linux Server Monitor

A simple yet powerful **Linux server monitoring script** to track system health including CPU, memory, disk usage, and system uptime.

This project is built for learning **Linux scripting + DevOps fundamentals**.

---

## 🚀 Features

- 📊 CPU usage monitoring
- 💾 Memory usage tracking
- 🗄️ Disk space usage check
- ⏱️ System uptime display
- 📄 Simple shell script (lightweight & fast)
- 🧠 Easy to customize for automation

---

## 🛠️ Tech Stack

- Linux (Bash Shell)
- Shell Scripting
- Git & GitHub for version control
